# 🏥 Clinic Management System

A real-world **Clinic Management System** designed for healthcare operations, built with enterprise-level architecture using ASP.NET, VB.NET, and SQL Server.

---

## 🎯 Purpose of the System

This system was developed to replace manual clinic operations with a structured digital solution that improves:

- Patient data management
- Appointment scheduling efficiency
- Billing accuracy
- Administrative control
- Operational reporting

---

## 🧠 Key Features

### 👤 Patient Management
- Add / edit patient records
- Medical history tracking
- Patient search & filtering

### 📅 Appointment System
- Schedule and manage appointments
- Doctor availability handling
- Daily visit tracking

### 💰 Billing & Payments
- Invoice generation
- Payment tracking
- Financial summaries

### 👨‍⚕️ Doctors Module
- Doctor profiles
- Specialization management
- Schedule control

### 📊 Reporting System
- Daily clinic reports
- Revenue tracking
- Patient statistics

---

## ⚙️ Tech Stack

- ASP.NET WebForms / MVC
- VB.NET
- SQL Server
- JavaScript
- Bootstrap
- HTML / CSS
- IIS Deployment

---

## 🏗 Architecture

- Layered Architecture (Presentation / Business / Data)
- Scalable database design
- Role-based system structure
- Modular business logic separation
